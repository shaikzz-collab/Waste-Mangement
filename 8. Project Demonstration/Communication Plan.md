# Communication Plan

This document details the communication plan established to present the **WasteGuide AI** project to evaluators and project stakeholders.

---

## 1. Stakeholder Matrix

| Stakeholder | Role | Communication Channel | Frequency | Objective |
| :--- | :--- | :--- | :--- | :--- |
| **SmartBridge Evaluator** | Project Grading / Capstone Review | GitHub Repo / Video Demonstration | Submission | Assess AI code quality, UI, and documentation. |
| **Academic Mentors** | Technical Guidance | Slack Channels / Office Hours | Weekly Sprints | Code reviews and project advice. |
| **General Public** | End Users | Live Web App / Feedback form | Post-Launch | Collect user feedback on classification accuracy. |

---

## 2. Presentation Agenda

The final project review will be delivered via a recorded 5-minute video demonstration covering:
- **0:00 - 0:45**: Introduction, Problem Statement, and Slide Overview.
- **0:45 - 2:00**: Live Search & AI Scanning demonstration (showing classification outputs, step guides, and hazard alerts).
- **2:00 - 3:15**: Interactive Map walkthrough (filtering by category, selection lists, and showing the routing link).
- **3:15 - 4:15**: User History log search and the visual dashboard metrics (ChartJS terminal-themed displays).
- **4:15 - 5:00**: Technical Architecture summary (Flask, Firestore REST, Groq LLM API), offline fallbacks, and project scale directions.
