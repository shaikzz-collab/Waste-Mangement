# Screenshot Reference Guide

This document lists the essential user interface views you should capture as screenshots for your project submission slide decks and demo reports.

---

## Required Screenshots Checklist

### 1. The Home Search Page
- **Description**: The main entry terminal view showing the retro terminal title header and the quick selection grid.
- **Action to Capture**: 
  - Open `http://localhost:5173`.
  - Capture the top terminal header and the 12 quick-select nodes grid.
  - Save as `01_home_search.png`.

### 2. Recyclable Item Search Result
- **Description**: The classification result panel showing successful recycling steps.
- **Action to Capture**:
  - Type `cardboard box` in the search bar and hit Enter.
  - Scroll down to show the full `ResultCard` (green "Recyclable" badge, disposal notes, and numbered steps).
  - Save as `02_scan_recyclable.png`.

### 3. Hazardous Safety Alert Result
- **Description**: The safety warning card for batteries or toxic electronics.
- **Action to Capture**:
  - Search for `alkaline battery`.
  - Capture the red warning caution alert and the active "Hazardous: True" badge.
  - Save as `03_scan_hazardous.png`.

### 4. Interactive Geolocation Map
- **Description**: The recycling registry map with center coordinates.
- **Action to Capture**:
  - Navigate to the `MAP` page.
  - Click the `E-Waste` chip filter.
  - Capture the map container pins, the selected highlight CenterCard, and the accepted materials panel on the right.
  - Save as `04_map_registry.png`.

### 5. Performance Analytics Dashboard
- **Description**: The data charts compiled using ChartJS.
- **Action to Capture**:
  - Navigate to the `DASHBOARD` page.
  - Capture the 4 glowing StatCards and the 3 terminal-themed canvas charts (Line, Bar, Doughnut).
  - Save as `05_dashboard_analytics.png`.
